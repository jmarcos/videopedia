Exercícios de Dedução Natural

Dedução Natural para a Lógica Proposicional Intuicionista

Derivabilidade de sequentes

  • φ ∧ ψ ⊢ ψ ∧ φ

  • (φ ∧ ψ) ∧ δ ⊢ φ ∧ (ψ ∧ δ)

  • φ ⊢ φ ∧ φ

  • α ∧ β, γ ∧ δ ⊢ γ ∧ β

  • α → (β → γ) ⊢ β → (α → γ)

  • ⊢ α → (β → α)

  • ⊢ α → α

  • α → β ⊢ α → (α → β)

  • α → (α → β) ⊢ α → β

  • γ → α, γ → β ⊢ γ → (α ∧ β)

  • β ∨ (α ∧ β) ⊢ β

  • (α ∨ β) → γ ⊣⊢ (α → γ) ∧ (β → γ)

  • α ⊢ ¬¬α

  • β → α, β → ¬α ⊢ ¬β

  • α, ¬α ⊢ ¬β

  • α ∨ β, ¬α ∨ γ ⊢ β ∨ γ

  • α → β ⊢ ¬β → ¬α

  • ¬(α ∨ β) ⊣⊢ ¬α ∧ ¬β

Dedução Natural para a Lógica Proposicional Clássica

Derivabilidade de sequentes

  • Terceiro Excluído / Tertium Non Datur: ⊢ φ ∨ ¬φ

Tarefa: Demonstrar a mesma fórmula, invertendo a ordem de aplicação das regras de introdução da disjunção.

  • ¬¬α ⊢ α

  • ¬β → α, ¬β → ¬α ⊢ β

  • ¬α → ¬β ⊢ β → α

  • ¬(¬α ∨ ¬β) ⊢ α ∧ β

  • ⊢ (α → β) ∨ (β → α), via raciocínio por absurdo

  • ⊢ (α → β) ∨ (β → α), via terceiro excluído

Dedução Natural para a Lógica de Primeira Ordem

  • (∀x)(φ → ψ) ⊢ (∀x)φ → (∀x)ψ

  • (∀x)Q(x) ⊢ (∀y)Q(y)

  • (∀x₁)(∀x₂)R(x₁,x₂) ⊢ (∀x₂)(∀x₁)R(x₁,x₂)

  • (∀x)φ₁ ∧ φ₂ ⊣⊢ (∀x)φ₁ ∧ (∀x)φ₂

  • (∃x)P(x), (∀x)(∀y)(P(x) → Q(y)) ⊢ (∀y)Q(y)

  • (∀x)A(x), (∃y)(A(y) → B(y)), (∀z)(A(z) → C(z)) ⊢ (∃w)(B(w) ∧ C(w))

  • (∃x)(φ₁ ∨ φ₂) ⊣⊢ (∃x)φ₁ ∨ (∃x)φ₂

  • (∃x)(∀y)φ ⊢ (∀y)(∃x)φ

  • (∀x)¬φ ⊢ ¬(∃x)φ

  • ¬(∃x)¬φ ⊢ (∀x)φ

  • ⊢ (∃x)(∀y)(B(y) ∨ ¬B(x))

Derivabilidade de regras

  • Raciocínio por casos: Γ₁, ¬φ ⊢ ψ; Γ₂, φ ⊢ ψ / Γ₁, Γ₂ ⊢ ψ

  • Raciocínio por redução ao absurdo: Γ₁, ¬φ ⊢ ¬ψ; Γ₂, ¬φ ⊢ ψ / Γ₁, Γ₂ ⊢ φ

  • (≈sim) Γ ⊢ t₁ ≈ t₂ / Γ ⊢ t₂ ≈ t₁

  • (≈trn) Γ₁ ⊢ t₁ ≈ t₂; Γ₂ ⊢ t₂ ≈ t₃ / Γ₁, Γ₂ ⊢ t₁ ≈ t₃

Para reflexão

  • O que ocorre se, em vez de adicionarmos ao sistema de Dedução Natural para a Lógica Intuicionista a regra (⊥E_cls) Γ, ¬φ ⊢ ⊥ / Γ ⊢ φ, adicionarmos uma regra da forma Γ, ¬(α # β) ⊢ ⊥ / Γ ⊢ α # β para algum conectivo binário # da nossa linguagem?
  • O que ocorre se, em vez de adicionarmos a regra (⊥E_cls), adicionarmos a seguinte regra de consequentia mirabilis: (¬_cls) Γ, ¬α ⊢ α / Γ ⊢ α?
  • O que ocorre se, em vez de adicionarmos ao sistema de Dedução Natural para a Lógica de Primeira Ordem Intuicionista a regra (⊥E_cls), adicionarmos a regra (DNQ) Γ ⊢ (∀x)¬¬φ / Γ ⊢ ¬¬(∀x)φ?
