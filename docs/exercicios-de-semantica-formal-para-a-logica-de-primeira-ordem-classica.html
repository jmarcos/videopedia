Exercícios de semântica formal para a Lógica de Primeira Ordem Clássica

Validade global de sequentes

  • (∃y)(∀x)φ ▷ (∀x)(∃y)φ
  • (∀y)(∃x)φ ▶ (∃x)(∀y)φ
  • (∀x:P)φ ◁▷ ¬(∃x:P)¬φ
  • (∀x)(φ → ψ) ▷ (∀x)φ → (∀x)ψ
  • (∀x)φ → (∀x)ψ ▶ (∀x)(φ → ψ)

Validade de sequentes em interpretações particulares

  • w ≈ xf(ygz) ▷ᴵ w ≈ (xfy)g(xfz)

Correção de regras

  • Regras para o símbolo de igualdade
  • (∀E) Γ ⊢ (∀x)φ / Γ ⊢ φ[x ↦ t]
  • (∀I) Γ ⊢ φ[x ↦ y] / Γ ⊢ (∀x)φ
